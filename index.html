✅ Features

  • 🗓 Daily Grabovoi "Code of the Day"
  • 🔊 Female voice affirmations per category
  • 🖥 Windows 10 installer with shortcut
  • 🖨 Printable black & white ritual calendar

💻 How to Use

1. Install

Run installer/GrabovoiCalendarSetup.exe on Windows 10. Follow the prompts and open the app via desktop shortcut.

2. Use the Calendar

  • Launch the app
  • View the ritual and code for the day
  • Click a date for rituals and affirmations

3. Print Your Calendar

Open printable/Calendar.pdf and print it. Use it to track rituals, write notes, or display on your wall.

✏️ Customize or Build from Source

javac src/GrabovoiCalendarGUI.java
java -cp src GrabovoiCalendarGUI

You can edit Grabovoi codes, swap affirmations, or create your own version of the app if you know basic Java.
